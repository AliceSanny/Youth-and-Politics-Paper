This repository contains data and code for the article:  
*"Who Gets a Voice Online? Family Resources, Gender, and Cohort Differences in Youth Political Participation in Europe"*

## Structure
- `data/` : Dataset (.dta), questionnaire, codebook  
- `scripts/` : Stata `.do` files for analysis  

## How to reproduce
1. Load `dataset.dta` in Stata
2. Run `scripts/analysis.do`
